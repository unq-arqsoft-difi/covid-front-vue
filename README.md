## TL;DR

Ir a <https://github.com/unq-arqsoft-difi/covid-back-node.git> y levantar la API.

```sh
git clone https://github.com/unq-arqsoft-difi/covid-front-vue.git
cd covid-front-vue
npm install
npm run serve
```

La APP quedará levantada en `http://localhost:8080`
